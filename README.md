# CodeBlockDoc 
> Advanced Real-Time Technical Documentation Compiler for Engineering Teams.

CodeBlockDoc is an idea brought to reality which converts raw source scripts across multiple languages automatically into structures production-ready Markdown technical manuals, architectural parameter tables, functional execution flows, and targeted optimization reviews in under two seconds.

##  Production Tech Stack
- **Frontend Framework:** React (Vite) with Tailwind CSS v4 Architecture
- **Rendering Engine:** React Markdown & Lucide System Icons
- **Backend Architecture:** Node.js, Express, CORS
- **AI Core Intelligence:** Google Gemini 2.5 Flash (`@google/genai`)

## Core Features & Engine Layout
- **Instant Logic Block Ingestion:** Dynamically isolates functional boundaries, scopes, parameters, and variable arrays.
- **AST Structural Context Mapping:** Conceptually maps codebase logic into abstract layouts to pinpoint parameters and functional return objects seamlessly.
- **Granular Optimization Analysis:** Features dual-tab switching to instantly isolate structural technical debt, memory bottlenecks, or logical code improvements.
- **Clean-Tech Accessible UI:** Built with a premium, vibrant, minimalist glassmorphic workspace aesthetic for smooth developer experiences.
